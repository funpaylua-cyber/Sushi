# SUSHI UA — сайт-меню із замовленням

Сучасний демо-сайт для закладу доставки суші. Повне меню розбите по
вкладках, клієнт збирає страви в кошик, а сформоване замовлення
надсилається готовим текстом у **Telegram** (або копіюється в буфер).
Жодного бекенду — все працює як статичний сайт.

## Що всередині

- **`index.html`** — структура сторінки
- **`styles.css`** — стилі (темна елегантна тема, адаптив під мобільні)
- **`script.js`** — меню, вкладки, кошик і відправка замовлення

## Головні можливості

- 🎨 **Чорно-золотий преміум-дизайн** у стилі логотипа закладу
- 🗂️ **Меню по категоріях**: суші-бургер, Sushi Dog, філадельфії,
  каліфорнії, дракони, чіз роли, теплі роли, макі, закуски
- 🪟 **Картка товару** (як у топ-ресторанів): добавки (авокадо, ікра,
  соуси, імбир, васабі тощо) та блок «схожі страви»
- 🛒 **Кошик** з підрахунком суми, кількістю, добавками та видаленням
- 📋 Кнопка «Скопіювати замовлення» (готовий текст для Direct)
- ✈️ **Замовлення в Telegram** — підготовлено, вмикається одним прапорцем
  у `CONFIG.telegramEnabled` (зараз вимкнено)
- 📱 Повна адаптація під телефон

## Як налаштувати під себе

Відкрий `script.js` та зміни блок `CONFIG` на початку файлу:

```js
const CONFIG = {
  telegram: "sushiua_",     // ваш username у Telegram (без @)
  shopName: "SUSHI UA",
  telegramEnabled: false,   // поставте true, коли запустите онлайн-замовлення
};
```

Меню редагується в масиві `MENU` там само (назва, опис, ціна, категорія),
а добавки — у масиві `ADDONS`.

## Як подивитись

Просто відкрий `index.html` у браузері (подвійний клік) — нічого встановлювати
не потрібно. Сайт повністю статичний.

## Секції

- Шапка з навігацією та кнопкою «Замовити»
- Привітальний блок (hero) зі слоганом «Втілюємо найсмачніші бажання»
- Категорії: суші, сети, бургери, шаурма, піца, салати
- Меню з фільтрами та цінами
- Сети для компанії
- Переваги «Чому ми»
- Блок замовлення + форма заявки
- Відгуки гостей
- Підвал з контактами, графіком і соцмережами
