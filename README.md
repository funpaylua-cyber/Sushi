# SUSHI UA — макет сайту

Сучасний демо-макет односторінкового сайту для закладу доставки їжі
(суші, бургери, шаурма, піца, салати). Зроблений як приклад того, як може
виглядати сайт замість/на додачу до Instagram-профілю.

## Що всередині

- **`index.html`** — структура сторінки
- **`styles.css`** — стилі (темна елегантна тема, адаптив під мобільні)
- **`script.js`** — меню з цінами, фільтри категорій, мобільне меню, анімації

## Як подивитись

Просто відкрий `index.html` у браузері (подвійний клік) — нічого встановлювати
не потрібно. Сайт повністю статичний.

## Секції

- Шапка з навігацією та кнопкою «Замовити»
- Привітальний блок (hero) зі слоганом «Втілюємо найсмачніші бажання»
- Категорії: суші, сети, бургери, шаурма, піца, салати
- Меню з фільтрами та цінами
- Сети для компанії
- Переваги «Чому ми»
- Блок замовлення + форма заявки
- Відгуки гостей
- Підвал з контактами, графіком і соцмережами

## Що варто замінити на справжнє

- Емодзі-плейсхолдери → реальні фото страв
- Телефон, адреса, посилання на Instagram/Telegram/Viber
- Ціни та склад меню під актуальне
- Лого / назву закладу

> Це демонстраційний макет для презентації ідеї.
